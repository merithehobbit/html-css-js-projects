# Filter Fun App

A simple browser-based project that lets users apply visual filters and effects to content in real time. The app is built with basic web technologies and focuses on interactive UI updates using JavaScript, HTML, and CSS.

## Features

- Real-time filter adjustments
- Simple and responsive interface
- Lightweight and fast performance
- Easy-to-use controls for visual changes
- Instant preview of applied effects

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

No build tools or installations required.
